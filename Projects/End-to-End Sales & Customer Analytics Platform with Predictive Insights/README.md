# 📊 Sales & Customer Analytics Platform

## 🚀 Project Overview

This project is an end-to-end Sales & Customer Analytics solution built using Python, Machine Learning, and Power BI.

It helps analyze:

* Sales performance
* Customer behavior
* Churn prediction

---

## 🛠️ Tech Stack

* Python (Pandas, NumPy, Scikit-learn)
* SQL (SQLite)
* Power BI

---

## 📂 Project Structure

sales-analytics-platform/
├── data/
├── scripts/
├── sql/
├── powerbi/
├── models/
└── Sales_Analytics_Platform.pbix

---

## 📈 Features

* Synthetic dataset generation using Python
* Churn prediction model using Random Forest
* Customer segmentation using RFM metrics
* Interactive Power BI dashboard
